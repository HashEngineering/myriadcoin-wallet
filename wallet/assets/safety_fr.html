Quelques mots sur la sécurité de vos Myriadcoins:

  • Les Myriadcoins sont stockés sur l'appareil avec Myriadcoin Wallet. Si vous le perdez, vous perdez vos Myriadcoins.
  • Cela signifie que vous avez besoin de sauvegarder vos clés privées! Conservez-les dans un endroit sécurisé et souvenez-vous de votre mot de passe.
  • Avant de désinstaller Myriadcoin Wallet (ou d'effacer les données de l'application/effacer les données de votre appareil), transférez vos Myriadcoins dans un autre porte-monnaie. Tout les Myriadcoins restant seront perdus.
  • Les paiements sont irréversibles. Si vous envoyez vos Myriadcoins dans le vide, il n'y a aucun moyen de les récupérer sauf si vous connaissez le destinataire et qu'il est d'accord pour vous les renvoyer.
  • Gardez votre appareil mobile en sécurité! N'installez que des applications de confiance. Des applications malveillantes pourraient essayer de voler votre porte-monnaie.
  • Minimisez les risques! Utilisez Myriadcoin Wallet seulement pour des petits montants pour un usage quotidien.
